# 考試範圍認知深度索引圖

> 用地理 agent 教的「題型-分值-認知層次矩陣」方法，把所有考試對應的複習資料按認知層次標記出來。
>
> 🟢 recall = 知道事實 &nbsp;&nbsp; 🟡 understand = 理解 &nbsp;&nbsp; 🟠 explain = 解釋 &nbsp;&nbsp; 🔵 analyze = 分析 &nbsp;&nbsp; 🟣 compare = 比較 &nbsp;&nbsp; 🔴 evaluate = 評價

---

## 如何用這份索引

每次複習一個主題，先看這個表：
1. 看你現在到哪個層次（recall → understand → explain → analyze → compare → evaluate）
2. 對照模擬題中該主題的題目，看認知層次是否匹配
3. 如果考試會考 evaluate 但你只複習到 recall → 去翻對應的技能框架文件

## 複習策略建議

### 按認知層次的時間分配

```
recall（知道）：    20% 時間  ← 你現在花了 80% 時間在這裏
understand（理解）：15% 時間
explain（解釋）：   20% 時間  ← 考試會在這裏考你
analyze（分析）：   20% 時間  ← 資料題的核心
compare（比較）：   10% 時間  ← 拉開分數的關鍵
evaluate（評價）：  15% 時間  ← 最高分題型
```

### 針對性建議

| 如果你…… | 建議 |
|---------|------|
| recall 層次還不穩 | 先背熟`考試範圍_集中溫習.md`的重點，再用`互動溫習工作紙.md`自檢 |
| recall 已經很熟 | 不要重複背了！轉到 **explain→analyze**——用`因果鏈分析框架.md`練習解釋「為甚麼」 |
| 想挑戰高分 | 重點做**compare 和 evaluate**——用`對比分析框架.md`和`評價分析框架.md`練習開放題 |
| 資料題不會答 | 先學`史料分析框架.md`的 S-T-A-R 四問法，然後做模擬題 Q5 的資料題 |
| 模擬題覺得太淺 | 用新版模擬題測試——資料題全部要求「引用證據」，close book 計時做 |

### 學習路徑建議

```
先：考試範圍.md（事實庫）→ 互動溫習工作紙.md（理解+自檢）
   ↓
再：因果鏈分析框架.md（解釋原因）→ 對比分析框架.md（比較異同）
   ↓
然後：史料分析框架.md（資料題方法）→ 評價分析框架.md（開放題方法）
   ↓
最後：模擬試卷.html（綜合測試，close book 90分鐘）
```
