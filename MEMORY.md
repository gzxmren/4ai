## 做了什么
从地理 agent 的复盘经验出发，对中国历史科复习资料做了「深度×广度×难度」三维改造。

### 诊断发现
- 原复习资料 ~80% 集中在 recall 认知层次
- 史料分析（过程性技能）完全缺失
- 模拟题资料题缺乏真实史料出处和 compare/evaluate 题目

### 4 个技能框架（新增）
| 文件 | 核心方法 | 解决什么问题 |
|------|---------|------------|
| 技能框架_史料分析.md | S-T-A-R 四问法 | 资料题不会分析史料 |
| 技能框架_因果链分析.md | 三步法 + 因果链图 | 解释题只知事实不知原因 |
| 技能框架_对比分析.md | 三步法 + 多维对比模板 | 比较题缺乏维度对齐 |
| 技能框架_评价分析.md | P-E-B-C 四步法 | 开放题立场模糊、论证单薄 |

### 改造文件
- **index.html**：从 3 个入口升级为 14 个入口的完整门户，每张卡片标注「原有/升级/新增/参考」
- **模擬試卷.html**：Section 5 重写——加入真实史料出处（马欢、严从简、李鴻章、康有为）、要求引用证据、新增 compare 和 evaluate 题目
- **模擬試卷_答案.html**：新增评分标准 + 答题框架引用提示
