# 對比分析框架

> 歷史考試常問「比較 X 和 Y 的異同」。比的不只是「知道」，而是 **能否在同一個維度上做對比**。

---

## 對比分析三步法

### Step 1 — 選取維度（Dimensions）

比較的第一步不是「找不同」，而是 **確定從哪些方面去比較**。

常用的比較維度：

| 維度 | 適合比較的對象 |
|------|--------------|
| 時間/年代 | 任何歷史事件 |
| 動機/目的 | 人物、政策、運動 |
| 規模/範圍 | 事件、戰爭 |
| 手段/方法 | 改革運動、治理方式 |
| 結果/影響 | 政策、條約、戰爭 |
| 深度/程度 | 改革、變法 |
| 性質 | 航海、戰爭、貿易 |

### Step 2 — 填內容（Data）

在選定的維度下，分別列出 X 和 Y 的相應內容。

### Step 3 — 下結論（Conclusion）

最後總結一句話，指出 **最關鍵的差異** 或 **根本的相同點**。

## 中二歷史必考對比表

### 對比一：鄭和 vs 哥倫布

| 維度 | 鄭和 | 哥倫布 | 異/同 |
|------|------|--------|-------|
| 時間 | 1405-1433 | 1492 起 | 鄭和早近 90 年 |
| 規模 | 寶船長138米，2-3萬人 | 船∼30米，約百人 | 鄭和遠大於哥倫布 |
| 航程 | 到非洲東岸 | 到美洲 | 方向不同 |
| 動機 | 宣揚國威、朝貢貿易 | 殖民擴張、掠奪資源 | 性質完全不同 |
| 結果 | 後被停止 | 歐洲各國爭相跟進 | 命運相反 |

**一句話結論**：鄭和航海是「政治秀」，哥倫布航海是「殖民擴張」——性質決定了命運。

---

### 對比二：懷柔政策 vs 高壓政策（清初對漢人）

| 維度 | 懷柔（胡蘿蔔） | 高壓（大棒） |
|------|--------------|-------------|
| 目的 | 爭取漢人支持 | 壓制反抗 |
| 措施 | 禮葬明思宗、開科取士 | 剃髮令、圈地、滿漢不通婚 |
| 效果 | 吸引讀書人合作 | 激起反抗但最終鎮壓 |
| 適用對象 | 文人、官僚 | 普通百姓 |

---

### 對比三：洋務運動 vs 戊戌維新 vs 清末新政

| 維度 | 洋務運動 | 戊戌維新 | 清末新政 |
|------|----------|----------|----------|
| 時間 | 1861-1895 | 1898（103天） | 1901-1911 |
| 觸發 | 鴉片戰爭+太平天國 | 甲午戰敗 | 八國聯軍 |
| 態度 | 主動學習 | 主動求變 | 被迫改革 |
| 範圍 | 局部（軍事、工業） | 全面（含政治） | 全面（含預備立憲） |
| 手段 | 中體西用 | 全變速變 | 妥協緩慢 |
| 阻礙 | 守舊勢力 | 慈禧發動政變 | 革命派取代 |
| 結果 | 甲午戰敗證明失敗 | 百日維新被推翻 | 加速革命→清朝亡 |

**一句話結論**：從洋務到維新再到新政，改革的深度逐步加深，但速度永遠跟不上革命的腳步。

### 對比四：兩個不平等條約

| 維度 | 《南京條約》(1842) | 《馬關條約》(1895) |
|------|------------------|------------------|
| 戰爭 | 第一次鴉片戰爭 | 甲午戰爭 |
| 對手 | 英國 | 日本 |
| 賠款 | 2100萬兩 | 二億兩 |
| 割地 | 香港島 | 台灣、澎湖 |
| 通商 | 五口通商 | 增開更多口岸 |
| 意義 | 第一個不平等條約 | 標誌洋務運動失敗 |

## 如何用對比框架答題

> **問題**：比較洋務運動和戊戌維新，你認為哪一個更有可能成功？試解釋。

**範例答題結構**：

| 答題部分 | 內容 |
|---------|------|
| ① 選維度 | 從動機、範圍、手段、結果四個維度比較 |
| ② 列內容 | （見上方對比表） |
| ③ 分析 | 洋務只學技術不改制度 → 甲午戰敗證明失敗 |
| ④ 分析 | 維新全面改革 → 但太快太急 → 守舊勢力反撲 |
| ⑤ 結論 | 兩者各有侷限：洋務太淺、維新太急。成功需要「深度合適 + 時機恰當」 |

## 實戰練習

### 練習一：填充對比表

比較「第一次鴉片戰爭」和「第二次鴉片戰爭」：

| 維度 | 第一次鴉片戰爭 | 第二次鴉片戰爭 |
|------|---------------|---------------|
| 時間 | 1840-1842 | |
| 對手 | | 英法聯軍 |
| 主要條約 | | 《天津條約》《北京條約》 |
| 割地 | | 九龍半島南部 |
| 特殊事件 | | 火燒圓明園 |

### 練習二：自主設計對比

**問題**：比較康雍乾三位皇帝的施政特點。

**提示維度**：任期、主要貢獻、施政風格、歷史評價

### 練習三：開放式對比

**問題**：有人說「鄭和下西洋和哥倫布發現新大陸都是人類航海史上的偉大壯舉」，你同意嗎？試從動機和影響兩方面比較。

> **提示**：同意不同意都可以，關鍵是你在同一維度上做比較，並給出證據。
