## 第1天 | 明代部分 + 清朝開端

### 第一段 · 鄭和的生平與下西洋背景（30分鐘）

**學習內容：**
- 鄭和的身世：回回人賽典赤後裔，十一歲被俘入宮，燕王府立功
- 「鄭」姓的由來：鄭村壩之戰
- 下西洋的原因：
  - 明初經濟繁榮、國勢漸強
  - 宣揚國威、提升成祖威望
  - 拓展海外貿易
- 何謂「西洋」：南海以西至印度洋地區

**記憶要點：**
- 七次下西洋（1405-1433年）
- 最遠到達非洲東岸
- 寶船長138米，每次2-3萬人

---

### 第二段 · 鄭和下西洋的影響（30分鐘）

**學習內容：**
- 鄭和船隊路線圖：蘇州劉家港出發 → 南海 → 印度洋 → 波斯灣 → 非洲東岸
- 使團活動：宣讀詔書、貿易、平定海盜
- 影響：
  - 政治：宣揚國威，萬國來朝
  - 經濟：促進中外貿易
  - 文化：文化交流（如麒麟→長頸鹿、榴槤傳入）

**對比記憶：**
- 與西方哥倫布對比：鄭和船隊規模更大，時間更早
- 下西洋後來停止的原因：耗費巨大、朝貢貿易成本高

---

### 第三段 · 北京城的興建（30分鐘）

**學習內容：**
- 明成祖遷都北京的原因：
  - 地理形勢：北靠燕山、太行山，南控中原
  - 大運河連接南方
  - 北京是成祖「龍興之地」（燕王封地）
- 北京城的建置：
  - 元大都為基礎改建
  - 四重布局：宮城（紫禁城）→ 皇城 → 內城 → 外城
  - 「凸」字形布局

**記憶要點：**
- 北京城的傳說：「哪吒城」
- 正陽門是哪吒的腦袋

---

### 第四段 · 北京城的防禦與總結複習（30分鐘）

**學習內容：**
- 北京城的防禦設計：
  - 內城設防重點（高聳磚牆、箭樓、角樓）
  - 外城保護城南
- 明英宗時瓦剌進攻北京，于謙堅守

**本日小結複習：**
- 鄭和七下西洋的時間、路線、規模、影響
- 北京城四重布局示意圖（心算畫出來）
- 遷都北京的三個原因

---

### 第五段 · 清朝第1章開端：滿清入關（30分鐘）

**學習內容：**
- 女真三部：建州、海西、野人
- 努爾哈赤統一女真，1616年建立「後金」
- 八旗制度
- 皇太極改族名為「滿洲」，國號改為「清」
- 山海關之戰、吳三桂引清兵入關

**時間線記憶：**
```
1616 努爾哈赤建後金
1636 皇太極改國號為清
1644 清兵入關，定都北京
```

---

### 第六段 · 清初盛世：康雍乾（30分鐘）

**學習內容：**
- 清初統一的過程：消滅南明、平定三藩
- 康熙：擒鰲拜、平定三藩、收復台灣
- 雍正：攤丁入地、懲治貪官
- 乾隆：開拓疆土（新疆）、修《四庫全書》
- 「康雍乾盛世」：1662-1795年，134年全盛期
- 版圖擴張：勝於漢唐

**強記口訣：**
```
康熙平三藩，雍正攤丁賦，
乾隆征回部，盛世百三年。
```

## 第2天 | 清朝第1章後半 + 第2章全

### 第一段 · 清初民族政策（30分鐘）

**學習內容：**
- 對漢人政策：
  - 禮葬明思宗 → 爭取漢人支持
  - 開科取士 → 吸納漢人讀書人
  - 滿漢不通婚、圈地、剃髮令 → 高壓政策
- 對蒙古政策：聯姻、封爵、盟旗制度
- 對西藏政策：金瓶掣簽、駐藏大臣
- 對新疆政策：回部、大小和卓叛亂
- 改土歸流：西南少數民族地區

**核心對比：**
```
懷柔政策（科舉、禮葬）← 對漢人 → 高壓政策（剃髮、圈地）
```

---

### 第二段 · 清代中衰（30分鐘）

**學習內容：**
- 乾隆晚年：和珅專權、貪腐嚴重
- 「和珅跌倒，嘉慶吃飽」
- 白蓮教起事（1796-1804）：耗費銀兩二億兩
- 嘉慶以後國勢日衰的原因：
  - 吏治敗壞
  - 土地兼併
  - 人口增長快，耕地不足
  - 軍隊腐化

**自查問題：**
- 清代由盛轉衰的關鍵原因是什麼？
- 中衰表現在哪些方面？

---

### 第三段 · 西力東漸（30分鐘）

**學習內容：**
- 18世紀60年代：西方工業革命
- 清廷對外貿易政策：
  - 只開放廣州一口通商
  - 必須通過「公行」交易
- 馬戛爾尼使團（1793年）的要求被乾隆拒絕
- 阿美士德使團（1816年）再次被拒

**理解要點：**
- 工業革命使西方生產力大增
- 清廷閉關自守，對世界變化無知
- 中西方力量此消彼長

---

### 第四段 · 兩次鴉片戰爭（30分鐘）

**學習內容：**
- 背景：英國向中國大量走私鴉片，白銀外流
- 林則徐虎門銷煙（1839年）
- **第一次鴉片戰爭（1840-1842）：**
  - 《南京條約》1842年：割讓香港島、五口通商、賠款
- **第二次鴉片戰爭（1856-1860）：**
  - 英法聯軍火燒圓明園
  - 《天津條約》1858年
  - 《北京條約》1860年：割讓九龍半島南部

**不平等條約記憶：**
```
南京條約1842 → 割香港、五口通商
天津條約1858 → 增開商埠
北京條約1860 → 割九龍
```

---

### 第五段 · 太平天國運動（30分鐘）

**學習內容：**
- 背景：鴉片戰爭後賠款轉嫁百姓、廣西連年災荒
- 洪秀全：科舉失敗 → 創拜上帝會 → 金田起義（1851年）
- 定都天京（南京），頒布《天朝田畝制度》
- 天京事變 → 內部相殘 → 由盛轉衰
- 曾國藩湘軍、李鴻章淮軍、常勝軍（洋槍隊）
- 1864年天京陷落，太平天國滅亡

**影響記憶：**
1. 元氣大傷（死亡二千萬至五千萬人）
2. 曾李等漢人將領受重用
3. 催生洋務運動

---

### 第六段 · 甲午戰爭 + 八國聯軍（30分鐘）

**學習內容：**

**中日甲午戰爭（1894-1895）：**
- 背景：日本明治維新後向外擴張
- 朝鮮東學黨起義 → 中日出兵 → 戰爭爆發
- 黃海海戰：鄧世昌壯烈殉國
- 北洋艦隊全軍覆沒
- 《馬關條約》1895年：賠款二億兩、割讓台灣

**八國聯軍之役（1900）：**
- 義和團「扶清滅洋」
- 八國聯軍攻陷北京
- 《辛丑條約》1901年：賠款四億五千萬兩

**對比記憶：**
```
甲午戰敗 → 馬關條約 → 割台灣
八國聯軍 → 辛丑條約 → 巨額賠款
```

## 第3天 | 清朝第3章 + 總複習

### 第一段 · 洋務運動的背景與目標（30分鐘）

**學習內容：**
- 背景：兩次鴉片戰爭戰敗、太平天國刺激
- 口號：「師夷長技以制夷」（魏源提出）
- 曾國藩、李鴻章、左宗棠等推動
- 目標：自強、求富、育才

**洋務運動重要措施：**
```
軍事：江南製造局、北洋艦隊
實業：輪船招商局、開平煤礦
教育：同文館、派遣留學生
```

---

### 第二段 · 洋務運動的成就與局限（30分鐘）

**學習內容：**
- 成就：
  - 促進中外關係
  - 推動軍事近代化
  - 奠定工業基礎
  - 擴闊國人眼界
- 失敗原因：
  - 只學技術不學制度
  - 甲午戰敗標誌洋務運動失敗
  - 受守舊勢力阻撓

**記憶點：**
- 洋務運動是「中體西用」的實踐
- 只追求局部變革，軍事改革為主

---

### 第三段 · 戊戌維新（30分鐘）

**學習內容：**
- 背景：甲午戰敗後民族危機加深
- 康有為、梁啟超鼓吹變法
- 1898年光緒帝推行變法
- 重要措施：
  - 政治：裁撤閒散衙門、廣開言路
  - 經濟：設農工商總局、提倡實業
  - 軍事：新法練軍、武舉改試槍炮
  - 教育：改革科舉、設新式學堂
- 結果：慈禧太后發動政變，囚光緒，變法失敗

**比較：洋務 vs 維新**
```
洋務運動：只學技術，局部改革 → 甲午戰敗證明失敗
戊戌維新：全面整改，政治改革 → 守舊勢力反撲失敗
```

---

### 第四段 · 清末新政（30分鐘）

**學習內容：**
- 背景：八國聯軍後清廷威信掃地
- 新政內容：
  - 政治：預備立憲、廢除科舉
  - 經濟：設立商部、獎勵實業
  - 軍事：編練新軍（北洋新軍）
  - 教育：設立學堂、派遣留學生
- 結果：成立「皇族內閣」令立憲派失望
- 新軍支持革命 → 武昌起義導火線

**三場改革總結對比：**

| 項目 | 洋務運動 | 戊戌維新 | 清末新政 |
|------|----------|----------|----------|
| 時間 | 1861-1895 | 1898 | 1901-1911 |
| 程度 | 局部（軍事為主） | 全面（政治經濟軍事教育） | 全面（含預備立憲） |
| 態度 | 「中體西用」 | 全變、速變 | 被迫改革 |
| 結果 | 甲午戰敗失敗 | 百日維新被推翻 | 皇族內閣令人失望，加速革命 |

---

### 第五段 · 清廷圖強的其他內容 + 詹天佑（30分鐘）

**學習內容：**
- 詹天佑與京張鐵路（中國人自行設計建造的第一條鐵路）
- 「人」字形線路設計解決八達嶺坡度問題
- 改良自動掛鉤
- 「中國鐵路之父」

**梳理三場改革的因果關係：**
```
鴉片戰爭 → 洋務運動（學技術）
甲午戰敗 → 戊戌維新（學制度）
八國聯軍 → 清末新政（被迫全面改革）
新政失敗 → 革命思潮高漲 → 清朝滅亡
```
