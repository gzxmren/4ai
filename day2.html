2 清初民族政策 ▶

「因俗而治」策略

The Qing ruled a multi-ethnic empire. They used different policies for different groups.
清朝統治多民族帝國，對不同民族用不同政策。

對漢人：一手胡蘿蔔，一手大棒

懷柔政策（胡蘿蔔）高壓政策（大棒）
禮葬明思宗 → 爭取漢人支持剃髮令 → 留頭不留髮
開科取士 → 吸納漢人讀書人圈地 → 強佔漢人土地
滿漢官員並用滿漢不通婚

對其他民族速記

民族管理方式一句話記
蒙古聯姻 + 盟旗制度嫁女兒、編旗子
西藏金瓶掣簽 + 駐藏大臣抽籤選活佛
西南少數民族改土歸流廢土司、派朝廷官
新疆回部武力平定不服就打
金瓶掣簽：達賴、班禪等活佛轉世時，把候選人名字放進金瓶抽籤決定。
改土歸流：廢除當地土司，改由朝廷派官直接治理。

3 清代中衰 ▶

由盛轉衰的關鍵

乾隆晚年 → 和珅專權 → 貪腐嚴重 → 國庫空虛

>

「和珅跌倒，嘉慶吃飽」— 和珅家產相當於清朝十多年財政收入！

四大中衰原因

原因說明一句話記
①吏治敗壞和珅帶頭貪，下面跟著貪上樑不正下樑歪
②土地兼併有錢人買光地，農民沒地種富者田連阡陌
③人口爆炸耕地跟不上人口增長人多地少吃不飽
④軍隊腐化白蓮教起事暴露軍隊無能兵不能打仗了
白蓮教起事（1796-1804）：民間宗教組織發動，歷時九年，清廷花了二億兩白銀才平定，暴露了清朝國力下降。

4 西力東漸 ▶

背景：工業革命

18世紀60年代，英國開始工業革命。
Machine replaced handmade. Production exploded.
機器取代手工，生產力爆炸式增長。

>

英國需要：①原料（棉花、絲綢）②市場（賣商品）

清廷的反應：閉關

政策內容
一口通商只開放廣州一個港口
公行制度外商必須通過指定的商行交易
限制外商只能在廣州特定區域活動，不能學中文

兩次錯過的機會

使團年份要求結果
馬戛爾尼1793通商、設使館被乾隆拒絕
阿美士德1816同被嘉慶拒絕
後果：英國轉向走私鴉片 → 白銀外流 → 鴉片戰爭

5 兩次鴉片戰爭 ▶

時間線

條約速記表

條約年份戰爭主要內容一句話記

火燒圓明園：1860年英法聯軍攻入北京，放火燒毀圓明園——世界園林藝術的奇蹟。

6 太平天國運動 ▶

時間線速記

1851 金田起義 → 1853 定都天京 → 1856 天京事變 → 1864 天京陷落

洪秀全的一生（一句話版）

科舉落榜 → 創拜上帝會 → 金田起義稱天王 → 定都南京 → 天京事變內部互殺 → 病逝 → 天國滅亡

影響（三個關鍵）

影響說明
①元氣大傷死亡2000-5000萬人，江南經濟崩潰
②漢人將領抬頭曾國藩、李鴻章因鎮壓有功，被重用
③催生洋務運動洋槍隊讓清廷見識到西方武器威力
天京事變：太平天國內部權力鬥爭，楊秀清、韋昌輝等互相殘殺，運動由盛轉衰的轉折點。
