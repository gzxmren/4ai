# 因果鏈分析框架

> 歷史考試最常問「為甚麼會這樣？」和「帶來了甚麼影響？」。不是背答案，而是會搭建因果鏈。

---

## 因果鏈三步法

### Step 1 — 定位事件（What）
確定你要分析的是哪個歷史事件。

### Step 2 — 往前找原因（Why）
問：**這件事為甚麼會發生？**

| 原因類別 | 思考方向 | 例子（鴉片戰爭） |
|---------|---------|----------------|
| **遠因（長期）** | 結構性問題、長期矛盾 | 清廷閉關政策、英國工業革命 |
| **近因（短期）** | 直接觸發點 | 林則徐虎門銷煙 |
| **根本原因** | 最深層的結構性原因 | 中西方力量此消彼長 |
| **導火線** | 最後一根稻草 | 英國國會通過對華出兵議案 |

### Step 3 — 往後推影響（So What）
問：**這件事帶來了甚麼後果？**

| 影響類別 | 思考方向 | 例子（《南京條約》） |
|---------|---------|-------------------|
| **直接影響** | 立刻發生的後果 | 割讓香港島、五口通商 |
| **間接影響** | 連鎖反應 | 白銀外流、社會矛盾加劇 |
| **短期影響** | 幾年內 | 英國在華勢力擴大 |
| **長期影響** | 幾十年後 | 中國逐步淪為半殖民地 |

## 答題示範

> **問題**：試分析清廷「閉關政策」如何導致了鴉片戰爭的爆發？

**範例答案**：

| 答題部分 | 內容 |
|---------|------|
| ① 閉關政策的內容 | 清廷只開放廣州一口通商，並限制外商活動 |
| ② 後果一 | 中英貿易失衡——中國大量出口茶葉絲綢，英國卻找不到商品平衡 |
| ③ 後果二 | 英國轉向走私鴉片，白銀大量外流 |
| ④ 觸發事件 | 林則徐虎門銷煙，英國找到出兵藉口 |
| ⑤ 結論 | 閉關政策製造了貿易矛盾，間接導致了戰爭爆發 |

> **答題結構**：政策內容 → 直接後果 → 連鎖反應 → 終點事件

## 實戰練習

### 練習一：補完因果鏈

完成以下因果鏈的空格：

```
英國走私鴉片 → [   ] → 銀貴錢賤 → 百姓負擔加重
                          ↓
                     [   ] → 皇帝派林則徐禁煙
                          ↓
                     虎門銷煙 → [   ]
```

> **答案**：白銀外流 → 道光皇帝憂慮 → 英國出兵

### 練習二：按要求搭建因果鏈

**問題**：試用因果鏈解釋為甚麼洋務運動會失敗。

**參考答案**：

```
根本問題：只學技術不改制度（中體西用）
    ↓
軍事建設：北洋艦隊亞洲第一
    ↓
但體制不變：官員貪腐、管理混亂
    ↓
甲午戰爭 → 北洋艦隊全軍覆沒
    ↓
洋務運動失敗的證明
```

### 練習三：多因素因果

**問題**：清朝由盛轉衰的原因有哪些？把它們按照重要程度排序。

**框架**：
1. 吏治敗壞（和珅專權）→ 根本原因
2. 土地兼併 → 社會不穩
3. 人口爆炸 → 資源不足
4. 軍隊腐化 → 無法應對內亂

> **答題提示**：排序題沒有標準答案，關鍵是你能否說出「為甚麼你認為這個更重要」。
