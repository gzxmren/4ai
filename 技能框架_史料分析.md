# 史料分析框架

> 考試中，史料分析題通常佔 15-40% 的分數。這類題目不是考你「記不記得」，而是考你「會不會用史料作為證據來回答問題」。

---

## 第一步：辨別一手史料 vs 二手史料

| | 一手史料（Primary Source） | 二手史料（Secondary Source） |
|---|---|---|
| **定義** | 事件發生當時產生的記錄 | 後人根據一手史料寫成的著作 |
| **例子** | 奏摺、詔書、信件、日記、當時的畫作 | 教科書、學者論文、歷史著作 |
| **優點** | 最接近事實，可信度高 | 有分析和總結，易於理解 |
| **局限** | 可能有偏見、遺漏 | 可能受作者觀點影響 |

### 中二歷史常見一手史料類型

| 史料類型 | 示例（本課） | 分析重點 |
|---------|-------------|---------|
| 奏摺/奏疏 | 大臣上奏反對下西洋 | 作者的立場和目的 |
| 詔書 | 明成祖詔書 | 統治者的官方說法 |
| 出使記錄 | 馬歡《瀛涯勝覽》 | 親歷者的觀察 |
| 條約原文 | 《南京條約》條款 | 哪一方獲利 |
| 統計數據 | 賠款金額、人口 | 數據說明了甚麼趨勢 |

## 第二步：史料分析四問法（S-T-A-R）

每次看到一段史料，按以下順序問自己四個問題：

### S — Source（來源）
- 這份史料是誰寫的/製作的？
- 是甚麼時候寫的？
- 是官方文件還是個人記錄？

### T — Tone（立場/語氣）
- 作者支持誰？反對誰？
- 用了哪些帶有感情色彩的詞？
- 作者的目的是甚麼？
  - 記錄事實？說服別人？讚揚？批評？

### A — Argument（論點）
- 這段史料的核心訊息是甚麼？
- 它想證明甚麼？
- 它有沒有忽略某些信息？

### R — Reliability（可信度）
- 作者的立場會影響他的記載嗎？
- 有沒有其他史料可以交叉印證？
- 一手史料 ≠ 絕對可信（作者也有偏見）

> **口訣：看來源、看立場、看觀點、看可信**

## 第三步：用史料回答問題

考試常問的三種史料題：

| 題型 | 問法 | 答題要點 |
|------|------|---------|
| **① 直接提取** | 「根據資料一，指出……」 | 直接從資料抄錄關鍵詞 |
| **② 推論** | 「根據資料一並就你所知，解釋……」 | 資料 + 自己知識結合 |
| **③ 評價** | 「你是否同意資料二的觀點？試解釋。」 | 先表明立場，再用史料論證 |

### 答題範例

> **資料**：馬歡《瀛涯勝覽》記載：「鄭和船隊帶去絲綢、瓷器，換回香料、寶石。各國國王熱烈歡迎，爭相派遣使臣來華朝貢。」
>
> **問題**：根據以上資料，分析鄭和下西洋對中外關係的影響。

**答題示範**：
| 分析步驟 | 作答內容 |
|---------|---------|
| ① 史料類型 | 一手史料——馬歡是隨行人員，親眼所見 |
| ② 提取訊息 | 貿易往來（絲綢瓷器換香料寶石）、外交成果（各國遣使朝貢） |
| ③ 推論 | 促進了中外貿易和文化交流，增強了明朝的國際影響力 |
| ④ 限制定位 | 馬歡是明朝官員，記載可能傾向正面，忽略了朝貢貿易的財政負擔 |

## 第四步：實戰練習題

### 練習一：史料辨識

判斷以下各項屬於一手史料還是二手史料：

1. 《南京條約》原文
2. 教課書對鴉片戰爭的描述
3. 林則徐寫給道光皇帝的奏摺
4. 中學歷史網站的介紹文章
5. 《明史·鄭和傳》

> **答案**：①一手 ②二手 ③一手 ④二手 ⑤二手（明史是清朝編寫的）

### 練習二：史料分析（S-T-A-R）

> **資料**：嚴從簡《殊域周咨錄》記載（明朝官員）：「（鄭和）費了數千萬錢糧，數萬計的性命……雖然帶了不少珍異寶物回國，但又有甚麼益處呢？」

用 S-T-A-R 框架分析這份史料：

| 步驟 | 你的分析 |
|------|---------|
| **S（來源）** | 明朝官員嚴從簡，寫於明朝中後期 |
| **T（立場）** | 反對下西洋——強調花費巨大、傷亡慘重 |
| **A（論點）** | 鄭和下西洋得不償失，浪費國力 |
| **R（可信度）** | 他是明朝人，對當時狀況有了解，但立場鮮明（反對派），可能誇大了負面影響。需對比馬歡等親歷者的正面記載 |

### 練習三：史料回答

> **資料**：1860年《北京條約》條款：「將粤東九龍司地方一區，歸英國屬香港界內。」
>
> **問題**：根據以上資料，指出第二次鴉片戰爭後英國在中國獲得了甚麼利益？

**參考答案**：
- 直接提取：英國取得了九龍半島南部（九龍司）
- 推論：英國在中國的佔地面積擴大，由香港島擴展至九龍半島
- 背景連結：結合 1842 年《南京條約》割讓香港島，英國在華勢力範圍持續擴大
