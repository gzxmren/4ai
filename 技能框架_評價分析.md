# 評價分析框架

> 評價題是歷史考試中認知層次最高的題型。不是考你「記不記得」，而是考你「會不會思考和判斷」。
>
> 評價 = 立場 + 論證 + 平衡

---

## 評價答題四步法（P-E-B-C）

### Step 1 — Position（表立場）
明確說出你的觀點：同意/不同意/部分同意。
> **範例**：「我同意這個觀點，因為……」或「我部分同意，因為……」

### Step 2 — Evidence（給證據）
用具體史實支持你的觀點。不要空談。
> **範例**：「以鄭和為例，他的船隊有寶船長 138 米，每次 2-3 萬人……」

### Step 3 — Balance（做平衡）
承認另一個角度的合理性。這是最能拉分的部分。
> **範例**：「然而，也有觀點認為下西洋花費巨大……」

### Step 4 — Conclusion（下結論）
綜合正反兩面，給出最終判斷。
> **範例**：「總體而言，我認為……」或「雖然有負面影響，但正面意義更大……」

> **口訣：先說你站哪，再用證據撐，反面也考慮，最後下結論。**

## 中二歷史常見評價題

| 評價對象 | 常見問題 | 思考方向 |
|---------|---------|---------|
| 鄭和下西洋 | 「鄭和下西洋是成功還是失敗？」 | 正面：宣揚國威/促進貿易；反面：花費巨大/得不償失 |
| 閉關政策 | 「你認為閉關政策對清朝是好是壞？」 | 正面：維護穩定/減少衝突；反面：落後於世界/導致戰爭 |
| 洋務運動 | 「洋務運動為甚麼失敗？誰的責任？」 | 客觀：守舊勢力/制度問題；主觀：只學技術不學制度 |
| 戊戌維新 | 「維新派敗在哪裏？」 | 策略：太快太急；環境：守舊勢力太強；時機：不成熟 |
| 清末新政 | 「新政為甚麼反而加速清朝滅亡？」 | 太晚來不及/改革讓人更失望/培養了革命者 |

## 評價題通用模板

### 模板一：「是否同意」類

```
我（同意 / 部分同意 / 不同意）資料中的觀點。

一方面，我認為……是有道理的，因為……
（舉例論證 1-2 個史實）

另一方面，資料的觀點也有不足之處，因為……
（指出資料忽略的事實或另一面）

綜合以上分析，我（仍然同意 / 傾向不同意 / 認為各有利弊），
因為……（給出你的最終判斷）
```

### 模板二：「評價人物/事件」類

```
對（人物/事件）的評價應當一分為二來看。

積極方面：
- ______（事實一）
- ______（事實二）

消極方面：
- ______（事實一）
- ______（事實二）

總體而言，我認為______。
雖然有不足，但其（正面意義 / 歷史價值 / 深遠影響）不容忽視。
```

## 答題示範

> **問題**：「鄭和下西洋雖然聲勢浩大，但花費巨大，得不償失。」你同意這個觀點嗎？試解釋你的看法。（6 分）

**高分段答案示範**：

| 步驟 | 內容 |
|------|------|
| ① 表立場 | **部分同意**這個觀點。 |
| ② 正面證據 | 花費巨大是真實的：每次出動數百艘船、2-3 萬人，回賜遠多於貢品，朝貢貿易是「厚往薄來」。國家財政負擔沉重，這是後繼皇帝停止下西洋的直接原因。 |
| ③ 反面證據 | 但不能說「得不償失」。鄭和下西洋的政治和文化意義很大：宣揚了國威，三十多國來朝，促進了中外文化交流（如榴槤、長頸鹿傳入中國），推動了海上絲綢之路的發展。 |
| ④ 平衡論證 | 如果只算經濟帳，確實不划算；但如果算政治和文化帳，鄭和下西洋的歷史價值是巨大的。 |
| ⑤ 結論 | **綜合而言**，我認為這個觀點看到了問題的一面，但忽略了鄭和下西洋的長遠歷史貢獻。 |

> **評分**：立場明確 1 分 + 正面論證 2 分 + 反面論證 2 分 + 結論 1 分

---

> **問題**：「洋務運動完全失敗了。」你是否同意這個觀點？

**答題結構**：

| 步驟 | 內容 |
|------|------|
| ① 立場 | **部分同意**。 |
| ② 同意的一面 | 洋務運動的目標是「自強求富」，但 1895 年甲午戰爭北洋艦隊全軍覆沒，證明軍事近代化失敗。 |
| ③ 不同意的一面 | 但洋務運動也有成果：建立了第一批近代工業（江南製造局、輪船招商局）、培養了第一批留學生（詹天佑等）、擴寬了國人視野。為後來的改革奠定了基礎。 |
| ④ 結論 | 可以說洋務運動「目標失敗了，但客觀上推動了中國近代化」。 |

## 評價答題常見錯誤

| 錯誤 | 表現 | 正確做法 |
|------|------|---------|
| ❌ 只說立場沒論證 | 「我同意，因為確實是這樣」 | 一定要舉具體史實 |
| ❌ 只講一面 | 只說好處或只說壞處 | 正反兩面都要談 |
| ❌ 沒有結論 | 分析了一堆但沒給最終判斷 | 用「總體而言/綜合來說」總結 |
| ❌ 立場模糊 | 「好像有道理，又好像不對」 | 明確說出同意/不同意/部分同意 |

## 實戰練習

### 練習一：補全評價答案

> **問題**：你認為清廷的閉關政策正確嗎？

請使用 P-E-B-C 框架補全以下答案：

```
我認為清廷的閉關政策___________________________________。

從短期來看，閉關政策有合理之處：________________________。
________________________________________________________。

但從長期來看，閉關政策帶來了嚴重後果：____________________。
________________________________________________________。

綜合而言，我認為_________________________________________。
```

> **參考答案要點**：
> - 短期合理：維護邊境穩定、減少外來衝突、保持傳統秩序
> - 長期後果：錯過工業革命、閉目塞聽、導致落後挨打
> - 結論：短期有利長期有害，最終弊大於利

### 練習二：自主評價

**問題**：有人說「戊戌維新失敗是因為維新派太急了」，你同意嗎？試從不同角度分析。

**提示**：
- 維新派角度：形勢危急，不變法就會亡國，必須快
- 守舊派角度：改革觸動了太多人的利益，必然反撲
- 客觀角度：103天的改革涵蓋政治經濟軍事教育，確實太快，未能爭取足夠支持

### 練習三：比較評價

**問題**：比較洋務運動、戊戌維新、清末新政三場改革運動。你認為哪一場最有機會成功？為甚麼？

> **提示**：選任何一個都可以，關鍵是論證要充分。例如：
> - 洋務運動：最平穩但太淺，觸不到核心
> - 戊戌維新：最徹底但太急，無法落地
> - 清末新政：最全面但太晚，時不我與
